## Description

Example of inject theme and layout with next-mui-helper by using _app.js.

## How to use

[clone the repo](https://github.com/thundermiracle/next-mui-helper)

```sh
cd examples/withThemeApp
npm install    (yarn)
npm run dev    (yarn run dev)
```
