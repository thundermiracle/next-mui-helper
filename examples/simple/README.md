## Description

The simplest way of using next-mui-helper without inject material-ui's theme object.

## How to use

[clone the repo](https://github.com/thundermiracle/next-mui-helper)

```sh
cd examples/simple
npm install    (yarn)
npm run dev    (yarn run dev)
```
